# Convenções Git

- Mensagens de commit em **português** (pt-BR com acentuação).
- Prefixos: `feat:`, `fix:`, `refactor:`, `test:`, `docs:`, `proto:`, `chore:`.
- Exemplo: `feat: adicionar fluxo de verificação WhatsApp`
- Commits atômicos: 1 feature ou 1 fix por commit.
- Se tem mudanças de 2 tarefas, separar em commits distintos.
- Não usar `git add .` — adicionar arquivos específicos por nome.
- Não commitar `.env`, secrets ou credentials.
